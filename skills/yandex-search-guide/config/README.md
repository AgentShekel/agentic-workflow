# Настройка Yandex XML Search API

## Регистрация

1. Зайди на https://xml.yandex.ru/
2. Авторизуйся своим Яндекс-аккаунтом
3. Нажми "Начать работу" и подтверди регистрацию
4. На странице настроек скопируй:
   - **user** — твой логин Яндекса (e-mail или login)
   - **key** — API-ключ (длинная строка)

## Настройка

Создай файл `config/.env`:

```bash
YANDEX_XML_USER=ваш_логин
YANDEX_XML_KEY=ваш_api_ключ
```

Или установи переменные окружения:

```bash
export YANDEX_XML_USER=ваш_логин
export YANDEX_XML_KEY=ваш_api_ключ
```

## Проверка

```bash
bash scripts/quota.sh
```

Должно показать результат тестового запроса и "Connection OK".

## Лимиты

| Тип аккаунта | Зона | Лимит в сутки |
|--------------|------|---------------|
| Бесплатный | .ru (Россия) | 1000 запросов |
| Бесплатный | .com.tr (Турция) | 1000 запросов |
| Бесплатный | .com (весь мир) | 1000 запросов |

- Лимит обновляется в 00:00 по Москве
- Рекомендуемый интервал между запросами: 1 секунда
- Максимальная длина запроса: 400 символов, 40 слов
- Максимум результатов на странице: 100 (groups-on-page)

## Зоны поиска

- `yandex.ru/search/xml` — поиск по России (регионы РФ)
- `yandex.com.tr/search/xml` — поиск по Турции
- `yandex.com/search/xml` — мировой поиск

По умолчанию скилл использует `yandex.ru/search/xml`.
